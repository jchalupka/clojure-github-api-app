# Clojure Github API App

A library containing several utility functions to utilize the github api.

Usage:

| Input                         | Output                                      |
| ----------------------------- | ------------------------------------------- |
| $> lein run "flutter/flutter" | Table of the top 10 contibutors to flutter. |
